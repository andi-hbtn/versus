## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/versus.git
```

Navigate into the project:

```bash
cd versus
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
